# ai-study-plan
这是一个由浅入深学习ai的计划
## 🎯 最适合小白学习AI大模型和AI编程的项目推荐

根据最新信息，我为你整理了7个特别适合零基础初学者的优质GitHub项目：

### ⭐ **完全小白首选**

#### 1. **Generative AI for Beginners** (微软出品)
- **项目链接**：https://github.com/microsoft/generative-ai-for-beginners
- 📚 **特色**：专为零基础打造，涵盖大语言模型(LLM)、提示词设计、AI应用开发等
- ✅ **推荐理由**：微软官方维护，理论+实战结合，即学即用，无需编程基础

#### 2. **LLM-Universe** (Datawhale中文版)
- **项目链接**：https://github.com/datawhalechina/llm-universe
- 📚 **特色**：中文教程，从API调用、知识库搭建、RAG到实际应用全流程详解
- ✅ **推荐理由**：中文资料丰富，代码和实际场景紧密结合，特别适合做项目

---

### 📖 **进阶学习项目**

#### 3. **LLM Cookbook** (Datawhale)
- **项目链接**：https://github.com/datawhalechina/llm-cookbook
- 📚 **特色**：基于吴恩达课程的中文本地化版本，覆盖Prompt工程、API对接、模型微调
- ✅ **推荐理由**：免费Colab体验，适合国内用户，理论与实践兼顾

#### 4. **LLM-Course** (体系完整)
- **项目链接**：https://github.com/mlabonne/llm-course
- 📚 **特色**：系统化教程，难度阶梯分明，大量云端笔记本
- ✅ **推荐理由**：适合有一点编程基础、想深入理解原理的学习者

---

### 🚀 **动手实践项目**

#### 5. **Happy-LLM**
- **项目链接**：https://github.com/zjunlp/Happy-LLM
- 📚 **特色**：用最小模型让初学者在普通电脑也能跑通技术原理
- ✅ **推荐理由**：门槛极低，从实践做起，锻炼底层认知

#### 6. **LLM Action** (工程实践)
- **项目链接**：https://github.com/liguodongiot/llm-action
- 📚 **特色**：聚焦大模型工程化和落地应用，微调、推理、量化等
- ✅ **推荐理由**：想让项目上线落地的进阶学习者必备

#### 7. **MiniMind** (自己训练模型)
- **项目链接**：https://github.com/jingyaogong/minimind
- 📚 **特色**：最低2G显卡即可训练自己的小模型
- ✅ **推荐理由**：想从0开始训练模型的极客级项目

---

## 💡 **学习路线建议**

| 你的情况 | 推荐项目 |
|--------|--------|
| 🟢 完全零基础 | 先学 **Generative AI for Beginners** → 再学 **LLM-Universe** |
| 🟡 有点编程基础 | **LLM-Course** + **LLM Cookbook** + 小项目实战 |
| 🔴 想工程落地 | **LLM Action** + **Happy-LLM** 完整项目 |
| 🔵 想深度研究 | **MiniMind** 从头训练自己的模型 |

这些项目大多有中文文档或社区支持，都很适合自学和边做边学！👍
